# Portfolio - Justine Adjanou

Bienvenue sur mon portfolio personnel. Ce projet est mon "projet fil rouge" réalisé pendant ma formation en développement web.

## À propos du projet

Je suis Justine Adjanou, développeuse web en formation. À travers ce portfolio, je mets en pratique tout ce que j’apprends : HTML, CSS et JavaScript. 

L’objectif est de créer un site moderne, responsive et interactif qui présente mon parcours, mes projets et mes compétences.

## Fonctionnalités développées

- Design responsive et moderne avec Tailwind CSS
- Navigation mobile avec menu hamburger
- Générateur d’inspiration aléatoire
- Estimateur de temps de développement
- Galerie de projets avec système de filtres
- Section FAQ en accordéon
- Témoignages et présentation personnelle

## Technologies utilisées

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript (interactivité pure)
